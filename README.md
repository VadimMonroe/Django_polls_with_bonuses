<h1>Directions to Run</h1>

1. git clone https://github.com/VadimMonroe/Django_polls_with_bonuses.git

2. python -m venv venv

3. . venv/bin/activate

4. pip install requirements.txt

5. python manage.py runserver     

6. http://localhost:8000
